# POSIX utilities (Zig)
Utilities implemented as specified by [POSIX.1-2017](https://pubs.opengroup.org/onlinepubs/9699919799/utilities/contents.html)
in
[Zig](https://ziglang.org/).

## Compiling
Compile programs (default):
```sh
make progs
# or
make
```
Clean all:
```sh
make clean
```

## Style guide
Zig default style.
